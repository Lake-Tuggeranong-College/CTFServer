# Alarm Anomaly

This page describes the alarm anomaly challenge in the Cyber City CTF. The challenge involves analyzing a series of alarm logs to identify unusual patterns and determine the cause of the anomaly.

# Challenge Description

Participants will be provided with a set of alarm logs that contain various entries related to security events. The logs may include timestamps, event types, source and destination IP addresses, and other relevant information. The goal is to analyze the logs to identify any anomalies or suspicious activities that may indicate a security breach or attack.
